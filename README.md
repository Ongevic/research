# Victor Ongera Research Site

GitHub Pages site for Victor Nyabuti Ong'era, now configured to publish from the `research` repository at `https://ongevic.github.io/research/`.

## Structure

- `src/`: site pages, layouts, collections, assets, and content
- repo root: Jekyll config, GitHub workflow, and repository metadata

## Local development

Install dependencies:

```bash
bundle install
```

Run the site locally:

```bash
bundle exec jekyll serve
```
